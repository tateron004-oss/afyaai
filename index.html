Accessibility Checklist (ADA / WCAG-aligned)
What this demo already does
  • Keyboard focus styles + skip link
  • ARIA labels for key controls + aria-live for dynamic outputs
  • High-contrast dropdown and buttons

What you still must do for “real compliance”
  • Run a WCAG 2.1 AA audit (contrast, headings, landmarks, forms)
  • Screen reader tests (NVDA/JAWS/VoiceOver)
  • Keyboard-only test (no mouse) for every workflow
  • Caption policy for real video + multilingual captions
  • Alternative text for any real images/charts
